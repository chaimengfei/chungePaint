# 企业案例图片说明

## 图片放置位置
请将案例图片放在此目录下。

## 需要的图片文件

1. **yongwang.jpg** - 燕郊永旺项目图片
   - 路径：`/static/images/cases/yongwang.jpg`
   - 对应案例：燕郊永旺

2. **quanjude.jpg** - 北京全聚德项目图片
   - 路径：`/static/images/cases/quanjude.jpg`
   - 对应案例：北京全聚德烤鸭

## 图片要求
- 格式：JPG/PNG
- 建议尺寸：宽度 750px 以上，比例 16:9 或 4:3
- 文件大小：建议控制在 500KB 以内

## 使用方法
1. 将图片文件拖拽到此目录
2. 确保文件名与上述名称完全一致
3. 重新编译小程序即可看到效果
